# Security Policy

| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

To report a vulnerability: head to Issues, create an issue, and then
describe the problem you found. Describe what part/area/function of the code is 
vulnerable and/or accessible.

Once done, wait about 48 hours.
